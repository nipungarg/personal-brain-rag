# Personal Brain (RAG Vault)

A Retrieval-Augmented Generation (RAG) system that allows users to query
their personal documents and receive grounded answers with citations.

## Problem
LLMs cannot natively access or reason over private documents reliably.

## Solution
Use embeddings + vector search to retrieve relevant context and feed it
to an LLM at query time.
